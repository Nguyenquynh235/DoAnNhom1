📚 HỆ THỐNG QUẢN LÝ THƯ VIỆN UNETI - BẢN CHÍNH THỨC

✅ Sử dụng PHP + MySQL (XAMPP)
✅ Kết nối CSDL: quanly_thuvien (chuẩn theo sơ đồ)
✅ Giao diện đẹp, sáng, hiện đại - phân quyền rõ ràng

👤 Bạn đọc:
- Đăng ký / Đăng nhập
- Tìm kiếm sách
- Mượn / Trả sách
- Lịch sử mượn

🔐 Admin:
- Quản lý sách (upload ảnh)
- Quản lý bạn đọc
- Quản lý mượn / trả
- Thống kê sách mượn nhiều
- Sidebar hiện đại, layout gọn gàng

📂 Cấu trúc:
- auth/: đăng nhập, đăng ký, db.php
- index.php: trang bạn đọc
- muon_tra.php: quản lý mượn sách
- admin/: giao diện & chức năng quản trị
- includes/: header/footer cho layout
- uploads/: ảnh bìa sách
